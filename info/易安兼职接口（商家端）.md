# 商家
## 1. 发布兼职信息
#### 1.1 说明
#### 1.2 url 
post http://equator8848.xyz:8848/yian/merchant/pushJobInfo.do
#### 1.3 参数
- summary 兼职的简介 string
- jobTime 时间 string
- reward 兼职报酬 int
- rewardType 报酬的类型 string
- address 兼职的地点 string
- details 兼职的详细信息 string
- type 兼职的类型，如服务员之类的 string
- hireNum 招聘的人数 int
- school 兼职发布到哪个学校（只有这个学校的学生才可以看到） int
 #### 1.4 响应
response | status | msg | data
---|---|---|---
成功 |  1 | OK | null
失败 | -1 | ERROR | null

## 8. 修改商家个人联系信息
#### 8.1 说明
只修改联系信息，不涉及密码
#### 8.2 url
post http://equator8848.xyz:8848/yian/merchant/changeContactInfo.do
#### 8.3 参数
- phone string
- QQorWX string
- name string
- unit string
#### 8.4 响应
response | status | msg | data
---|---|---|---
成功 |  1 | OK | null
失败 | -1 | ERROR | null

接口统一返回格式 json
{
    status : xxx, （状态码，自定义的那种）
    msg : xxx, （简单的描述信息）
    data : xxx （数据）
}
状态码status为负数时代表失败，详细信息看msg
